KEYBOARD INPUT
	
ESC : 		zum Beenden das Programm sowohl im Menü als auch während des Spiels ohne zu speichern
Arrow down : 	der Spieler geht nach unten
Arrow up: 	der Spieler geht nach oben
Arrow left: 	der Spieler geht nach links
Arrow right: 	der Spieler geht nach rechts
Tab: 		das Spielmenü/Option offnen
F1 : 		die Legendentabelle offnen
F5 : 		zum Erfrischen das Terminal während des Spiels


HINWEIS : 
Falls die Monster oder der Spieler während des Spiels "Schatten" hinterlassen, d.h der Rechner läuft wegen der Threads langsamer als wenn es keine gibt und er kann nicht alle alten Zeichen von den beweglichen Objekten völlständig löschen, drucken Sie bitte F5 um das Terminal zu erfrischen. Danke!

Wenn der Player 1 Leben verliert, taucht er am Eingang wieder auf, die Monsters laufen weiter.

Jedes mal die Option "New game" ausgewählt wird, erstellt das Game ein neues Labyrinth in der Datei "level.properties"
